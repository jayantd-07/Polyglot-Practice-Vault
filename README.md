# Access Control Practice Program

Hey there! 👋 I'm currently learning C through CS50 (Week 2), and this is a small practice project I built to test my logic.

It's basically a mini "login system" (kinda like a game lobby). It asks for your Name, your Tech Tier (A or B), and a secret Access Code. Depending on the rules for each tier, it either grants or denies you access. 

### 🛠️ What I learned doing this:
* **`do-while` loops:** I used these to lock the user in until they give the correct input (so you can't type 'Z' when I only want 'A' or 'B').
* **Input Validation:** Making sure the program handles bad inputs without breaking.
* **Code Formatting:** I spent quite a bit of time fixing small spaces and indentations to make sure it passes Harvard's strict `style50` checks perfectly!

It's a simple program, but it really helped me get my hands dirty with C syntax and conditions.
